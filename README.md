# Snake Game 🐍

이 프로젝트는 HTML과 JavaScript로 만든 고전 Snake(스네이크) 게임입니다.

- 점수 랭킹(Top 5) 기록
- 속도 조절(Easy/Normal/Hard/Hell)
- 벽에 부딪히면 게임 오버
- 게임 오버 후에는 재시작 버튼으로만 재시작 가능

## 제작 도구
- Helpy chat(헬피 챗)과 Github MCP를 활용해 자동화된 방식으로 제작되었습니다.

## 실행 방법
1. 이 저장소를 클론하거나, 파일을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저로 열면 바로 게임을 플레이할 수 있습니다.

## GitHub Pages로 배포하는 방법
1. 이 저장소의 Settings(설정) > Pages 메뉴로 이동합니다.
2. "Branch"에서 `main` 브랜치와 `/ (root)` 폴더를 선택합니다.
3. 저장하면, 잠시 후 아래와 같은 주소에서 Snake 게임을 웹에서 바로 플레이할 수 있습니다:
   
   `https://{your-github-username}.github.io/snake-game-202406/`

> 예시: https://GeunCheolOh.github.io/snake-game-202406/
